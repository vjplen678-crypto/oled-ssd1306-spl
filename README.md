# STM32 OLED SSD1306 SPL Library

Custom OLED SSD1306 display driver library written using STM32 Standard Peripheral Library (SPL).

## Description
This project provides a lightweight SSD1306 OLED driver for STM32 microcontrollers using I2C communication. The library was developed from scratch using SPL APIs instead of HAL.

## Features
- SSD1306 initialization
- Text display
- Cursor positioning
- Clear screen
- Draw basic graphics / pixels
- I2C communication via STM32 SPL

## Technologies
- STM32
- Embedded C
- STM32 SPL
- I2C
- SSD1306 OLED Display

## Purpose
Used for embedded projects requiring simple display output such as menus, sensor values, and system status.

## Example Applications
- Sensor monitoring
- Menu interface
- Smart locker display
- Embedded debugging output
